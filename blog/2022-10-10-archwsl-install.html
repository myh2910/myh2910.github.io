Installing ArchWSL

2022-10-10

I wanted to execute bash in Windows, so I installed ArchWSL. Here are the steps.

  • First, install ArchWSL and unzip Arch.zip into a folder.
  • Execute Arch.exe and press Enter.
  • Execute Arch.exe again and follow the next steps.
    • Set root password by passwd.
    • Open /etc/sudoers and uncomment %wheel ALL=(ALL:ALL) ALL.
    • Add user by useradd -m -G wheel,audio,video {username}.
    • Set user password by passwd {username}.
  • Exit and run Arch.exe config --default-user {username} to set default user.
  • To initialize the keyring and use pacman, execute the following commands.
    • sudo pacman-key --init
    • sudo pacman-key --populate
    • sudo pacman -Sy archlinux-keyring
    • sudo pacman -Su
  • To check the WSL version, execute wsl -l -v.
  • To uninstall, run wslconfig.exe /unregister arch and remove the files.

See the ArchWSL documentation for more information.
